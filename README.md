# Ecommerce AI Voice Assistant

An advanced AI-powered voice assistant designed for **SAH Kitchen**, a commercial kitchen equipment provider. This system leverages **Vapi** for high-fidelity voice interactions and **n8n** for robust backend automation, integrating directly with **WooCommerce** and multiple AI models.

## 🚀 Overview

The assistant, named **Ryan**, acts as a senior kitchen consultant. It can handle complex B2B inquiries, provide technical equipment advice, check order statuses, and automate follow-up communications. By offloading these tasks to an AI, SAH Kitchen provides 24/7 expert support to its customers.

## ✨ Key Features

- **Expert Consultancy**: Real-time advice on commercial kitchen layouts, equipment selection, and operational efficiency.
- **WooCommerce Integration**: Direct access to real-time product data (pricing, stock, specs) and customer order tracking.
- **Smart Internet Search**: Integrated with Perplexity AI for live market research and technical data retrieval.
- **Automated Communication**: 
    - Drafts professional, personalized emails using Google Gemini.
    - Sends automated quotes and support tickets to the team.
    - Notifies suppliers for specific inquiries.
- **Human-in-the-Loop**: Escalates complex requests to human agents by creating support tickets.

## 🛠️ Tech Stack

- **Voice AI Platform**: [Vapi](https://vapi.ai/)
- **Automation Engine**: [n8n](https://n8n.io/)
- **E-commerce Platform**: WooCommerce
- **LLMs & Search**: 
    - Google Gemini 2.0 Flash (Email generation)
    - Perplexity AI (Internet search)
- **Programming**: Python (Data cleaning and processing in n8n)
- **Messaging**: SMTP for email delivery

## ⚙️ Setup & Installation

### 1. n8n Workflows
- Import the `.json` files from the `/workflow` directory into your n8n instance.
- Configure the following credentials in n8n:
    - **WooCommerce API**
    - **Perplexity API**
    - **Google Gemini API**
    - **SMTP Credentials**

### 2. Vapi Configuration
- Create a new Assistant in Vapi.
- Use the content of `vapi/master-prompt.txt` as the system prompt.
- Define tools in Vapi that point to the Webhook URLs generated by your n8n workflows.

### 3. Environment Variables
Ensure your n8n workflows are pointing to the correct WooCommerce store URL and have the necessary API keys configured for the AI models.
